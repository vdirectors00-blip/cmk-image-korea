# Sanity Studio 약관 2건 등록 가이드

> 이 가이드는 클라이언트 법무 검토 완료 후 약관 본문을 Sanity Studio에 등록하는 절차입니다.
> 등록 완료 시 cmkimage.co.kr/privacy · /terms 페이지에 본문이 자동 노출됩니다.

---

## 사전 준비

1. Studio 접속: `http://localhost:3333` (dev) 또는 배포된 Studio URL
2. 로그인: GitHub OAuth (V-Directors 계정)
3. 좌측 메뉴 → **페이지** (Page) 문서 유형 선택

---

## 등록 절차 — 개인정보처리방침

### 1단계: 새 문서 생성
- 페이지 목록 화면 우측 상단 **+ 새 문서** 클릭

### 2단계: 기본 정보 입력

| 필드 | 값 |
|------|----|
| **제목 (한국어)** | `개인정보처리방침` |
| **Slug** | Generate 버튼 클릭 → `개인정보처리방침` 생성됨 → **수동으로 `privacy` 로 변경** |
| **시행일** | 확정된 시행일 입력 (예: `2026-06-01`) |
| **버전** | `1.0` |

> ⚠️ **Slug 는 반드시 `privacy` 로 설정해야 합니다.** 사이트가 `/privacy` URL로 이 문서를 조회합니다.

### 3단계: 본문 입력

`docs/legal/개인정보처리방침_초안.md` 파일을 열고 내용을 PortableText 에디터에 구조별로 입력합니다.

**PortableText 에디터 활용법:**
- 각 `##` 제목줄 → 에디터에서 블록 유형을 **H2** 로 선택 후 제목 텍스트 입력
- 각 `###` 제목줄 → **H3** 선택
- 일반 문단 → **Normal** 유형
- 굵은 텍스트 → 텍스트 선택 후 **B** (강조) 클릭
- 표(Table) → PortableText 는 표를 직접 지원하지 않음. 표 내용은 `H3 + 목록(리스트)` 형태로 변환 권장

> ⚠️ **중요 — `rule.required()` 제약**: 본문 필드는 반드시 1줄 이상 입력해야 저장됩니다. 법무 검토 전이라면 임시 텍스트 `(준비 중)` 1줄이라도 입력 후 저장하세요. 빈 본문으로는 저장이 거부됩니다.

### 4단계: 저장
- 우측 상단 **게시(Publish)** 클릭 (초안 저장이 아닌 게시 필요)
- 사이트 `/privacy` 에서 확인

---

## 등록 절차 — 이용약관

1~4단계 동일. 아래 값만 변경:

| 필드 | 값 |
|------|----|
| **제목 (한국어)** | `이용약관` |
| **Slug** | **`terms`** 로 수동 설정 |
| **시행일** | 확정된 시행일 (개인정보처리방침과 동일 권장) |
| **버전** | `1.0` |

본문 소스: `docs/legal/이용약관_초안.md`

---

## 확인 방법

등록 완료 후:

1. dev 서버 실행 중이면 `http://localhost:4321/privacy` 접속
2. 상단에 PageHead (소문자 italic "privacy") + 시행일 + 본문 확인
3. 하단 Footer 에서 개인정보처리방침·이용약관 링크 클릭 → 정상 이동 확인
